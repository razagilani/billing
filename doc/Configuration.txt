Configuring File Server
=======================

predicate: Samba installed and running

mkdir /db/skyline/bills
sudo useradd -s /bin/true billing
sudo passwd billing
sudo smbpasswd -L -a billing
(smbpasswd matches linux passwd.  is linux passwd even needed?)


Update smb.conf

[db]
    path = /db/
    browseable = yes
    read only = no
    writeable = yes
    write list = 
    guest ok = no
    create mask = 0644
    directory mask = 0755
    force user = billing
    force group = billing

sudo service samba restart

Mounting this filesystem on a linux client: 

sudo smbmount //tyrell/db /db -o user=billing
